\entry{Unsupervised Skeleton-Based Action Segmentation via Hierarchical Spatiotemporal Vector Quantization}{2026}%
{arXiv 2026}{}
A hierarchical spatiotemporal vector quantization framework for unsupervised skeleton-based temporal
action segmentation, achieving SOTA on the HuGaDB, LARa, and BABEL benchmarks. \href{https://arxiv.org/abs/2604.15196}{View on arXiv}

\entry{Enhancing Human-Likeness in Reinforcement Learning Agents via Hierarchical Macro Action Quantization}{2026}%
{arXiv 2026}{}
A human-like RL framework that encodes human demonstrations into macro actions via a two-level
hierarchical macro action quantization approach (HiMAQ), applied to dexterous control of a humanoid
robot hand. On the D4RL benchmarks it outperforms the non-hierarchical MAQ baseline on human-likeness
while matching or exceeding success rates, generalizing across IQL, SAC, and RLPD.
\href{https://arxiv.org/abs/2604.15196}{View on arXiv}
